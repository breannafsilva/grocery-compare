# Future Enhancements

These are backlog notes for ideas that should be revisited after the core live
pricing and deployment flow are stable.

## Live Pricing

- Use the Kroger Locations API to resolve the right `locationId` from the ZIP
  code entered in the app instead of relying on a committed default.
- Try to connect live data for Jewel-Osco, including sale and digital coupon
  pricing if an approved Albertsons/Jewel source is available.
- Try to connect live data for Costco, including member pricing and warehouse
  offers if an approved provider or account integration is available.

## Design

- Apply the Vizient style guide once brand assets, tokens, typography, spacing,
  and accessibility guidance are available.

## Optimizer Experience

- Add sticky headers to wide optimizer tables so item and store context stays
  visible while scrolling.
- Add a sticky horizontal scrollbar for wide optimizer/price views so users do
  not need to scroll to the bottom before moving sideways.

## Customization

- Explore customizable widgets for dashboard panels and optimizer summaries.
  This is lower priority than live data quality and table ergonomics.
